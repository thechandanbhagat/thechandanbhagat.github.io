Quick Start

Up and Running in Seconds

Get started with ezPM2GUI in just a few simple steps

01

Install globally using npm

npm install -g ezpm2gui
02

Run from anywhere

ezpm2gui
03

Access the dashboard

Open your browser and navigate to http://localhost:5000

01

Install locally in your project

npm install ezpm2gui
02

Run using npx

npx ezpm2gui
03

Add to your package.json scripts

"scripts": { "pm2-ui": "ezpm2gui" }
01

Pull and run the Docker image

docker run -p 5000:5000 -v /path/to/pm2/data:/root/.pm2 thechandanbhagat/ezpm2gui
02

Using with docker-compose

services: ezpm2gui: image: thechandanbhagat/ezpm2gui ports: - "5000:5000" volumes: - /path/to/pm2/data:/root/.pm2

Compatible with all major operating systems - Linux, macOS, and Windows. Requires Node.js 14.x or higher.
